## `$ projects --featured`

<details>
<summary><strong>⚡ LiveSportz — Live Sports Commentary Engine</strong></summary>
<br>

A real-time sports commentary and data engine powered by WebSocket infrastructure. LiveSportz delivers live match updates, AI-generated commentary, and push-based score feeds to clients with sub-second latency. Built with a focus on concurrency, scalability, and resilience under high-traffic sports events.

| Attribute | Details |
|:---|:---|
| **Stack** | TypeScript · Node.js · WebSocket (ws) · Redis Pub/Sub · React |
| **Scale** | Designed for concurrent multi-match event streams |
| **Performance** | Sub-100ms event propagation via Redis pub/sub channels |
| **AI Layer** | LLM-powered dynamic commentary generation per game event |
| **Security** | Token-based WebSocket auth · rate limiting · event validation |
| **Impact** | Real-time sports experience with zero-polling architecture |
| **Repository** | [github.com/casanova259](https://github.com/casanova259) |

LiveSportz replaces polling-based sports dashboards with a true push model. Each game event — goal, foul, substitution — triggers an AI commentary chain and instantly propagates to all subscribed clients. The architecture is built for horizontal scale with Redis as the message broker between Node.js workers.

</details>
